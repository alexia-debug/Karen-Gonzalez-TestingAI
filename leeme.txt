Sistema de Gestión de Pacientes para Salud

 Descripción General
Este proyecto consiste en un sistema web de gestión de pacientes desarrollado como parte del proyecto final de QA 4.0.

 Interfaces Desarrolladas
- Formulario Registro de pacientes 
- Dashboard Principal: Vista general de actividades y estadísticas relevantes
- Perfil de Paciente: Información detallada y editable de cada paciente
- Gestión de Citas: Calendario interactivo para programación de consultas
- Gestión de Medicamentos: Registro y seguimiento de prescripciones


 Tecnologías y Herramientas Utilizadas
- Frontend: HTML5, CSS3, JavaScript
- Diagramas: Mermaid
- Pruebas Automatizadas: Selenium con c y NUnit, Playwright con ZeroStep y JavaScript en Cursor
- Diagramación: Mermaid para diagramas como código
- IA Generativa: Claude y Chat GPT

 Alteraciones
- Requerimientos: Se encuentran marcadas en rojo
- Casos de uso: Se encuentran marcadas en rojo
- Interfaces: Se encuentran dentro de la carpeta Código de la aplicación\Alteradas  (Las cuales incluyen, borrado de campos, eliminación de validaciones y errores de código)
- Diagramas: se encuentran en la carpeta Documentación\Diagramas con la inicial ALT
